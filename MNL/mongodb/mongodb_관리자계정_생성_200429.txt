1. /etc/mongod.conf 에서 authorization이 disabled 설정
========================================
--------------------------------------------------------------------------------
/etc/mongod.conf 파일에서 authorization을 disabled 설정한다.

/etc/mongod.conf 파일 백업
---------------------------------------
ls -al /etc/mongod.conf*

sudo cp -p /etc/mongod.conf /etc/mongod.conf.bk200429
---------------------------------------

-------------------------------------------------------------------------------
/etc/mongod.conf 파일 수정

---------------------------------------
sudo vi /etc/mongod.conf
---------------------------------------
- 수정 전
---------------------------------------
#security:
security:
#  authorization: 'disabled'
  authorization: 'enabled'
---------------------------------------
- 수정 후
---------------------------------------
#security:
security:
  authorization: 'disabled'
#  authorization: 'enabled'
---------------------------------------

-------------------------------------------------------------------------------
OS 재기동

sudo reboot

-------------------------------------------------------------------------------
관리자 계정 생성 후 다시 아래 값으로 원복한다.(보안 관련 설정)
authorization: enabled

1) 사용자 drop
---------------------------------------
