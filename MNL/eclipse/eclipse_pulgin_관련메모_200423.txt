## properties editor


이클립스에서 Properties 파일 편집을 하기 위해  PropEdit 또는 PropertiesEditor 라 불리는 플러그인이 있습니다. 설치방법은 Help > New Install Software... 선택후 Add..을 클릭하여 다음 주소를 추가합니다.

PropertiesEditor
http://propedit.sourceforge.jp/eclipse/updates/
http://propedit.sourceforge.jp/eclipse/updates
설치 항목이 나타나면
PropertiesEditor 항목을 체크하고 설치를 진행합니다.
(PropertiesEditor 만 체크 나머진 체크 해제)

properties editor, propertiesEditor_Asian_NLS 가 설치된다.

설치가 완료되면 .properties 파일 클릭후
open with > propertiesEditor가 나타나는지 확인한다.

### Installation
Install from the update site: https://takezoe.github.io/amateras-update-site/

For users who use Eclipse on Java 16 or later, add the following to eclipse.ini.

- java 16 이상 사용하는 Eclipse를 사용하면  eclipse.ini 파일에 아래 옵션을 추가해야 Amateras Modeler의 UML 파일이 생성된다.
	- java16 이상일 때 아래 옵션이 없으면 새파일 만들기가 안된다.

```
--add-opens=java.desktop/java.beans=ALL-UNNAMED
```
